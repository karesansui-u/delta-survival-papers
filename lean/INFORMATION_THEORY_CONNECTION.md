# Information Theory Connection: Three Directions
# 情報理論接続：3方向の数学的導出

## 概要

存続方程式 S = N_eff · (μ/μ_c) · e^{-δ} と情報理論の本質的接続を3方向で確立する。
これまでの論文では「語彙を共有するが数学的には未接続」だったものを、
定理レベルで接続する。

---

## Direction 1: δ ↔ KL Divergence の等価性

### 設定

- X_1,...,X_n ∈ {0,1} — Boolean変数（一様分布）
- C_1,...,C_m — 制約（個別充足確率 p_1,...,p_m）
- δ = Σ_i (-ln p_i) — 累積情報損失（ナット単位）
- P_0 = 2^{-n} · 1_{0,1}^n — 全割当上の一様分布
- P_SAT = |SAT|^{-1} · 1_SAT — 充足割当上の一様分布
- Z = |SAT| — 充足解の個数

### 定理 1.1（δ-KL恒等式：独立制約の場合）

制約が独立（変数を共有しない）のとき：

    D_KL(P_SAT || P_0) = δ    （恒等式）

**証明**: 一様分布間のKLダイバージェンスは：

    D_KL(P_SAT || P_0) = Σ_x P_SAT(x) ln(P_SAT(x) / P_0(x))
                       = Σ_x (1/|SAT|) · ln(2^n / |SAT|)
                       = ln(2^n / |SAT|)

独立制約のとき |SAT| = 2^n · Π_i p_i = 2^n · e^{-δ} が**正確に**成立。

したがって：
    D_KL = ln(2^n / (2^n · e^{-δ})) = ln(e^δ) = δ    ∎

### 定理 1.2（δ-KL不等式：一般の場合）

任意の制約構成に対して：

    E[D_KL(P_SAT || P_0)] ≥ δ

等号成立条件：Z = |SAT| が定数（分散0）のとき、かつそのときに限る。

**証明**: D_KL = ln(2^n / Z) = n ln 2 - ln Z

    E[D_KL] = n ln 2 - E[ln Z]

Jensen の不等式（ln は凹関数）より：

    E[ln Z] ≤ ln E[Z] = ln(2^n · e^{-δ}) = n ln 2 - δ

したがって：

    E[D_KL] = n ln 2 - E[ln Z] ≥ n ln 2 - (n ln 2 - δ) = δ    ∎

### 定理 1.3（ギャップの特性化：第二モーメントとの接続）

ギャップ E[D_KL] - δ は第二モーメント比 R_2 = E[Z²]/E[Z]² で特性化される：

    E[D_KL] - δ = ln E[Z] - E[ln Z]    （Jensen gap）

2次近似では：

    E[D_KL] - δ ≈ Var(Z) / (2 · E[Z]²) = (R_2 - 1) / 2

**意義**: これは前回形式化した第二モーメント法の結果を直接活用する：
- ペア相関 g(β) → E[Z²] → R_2
- R_2 が有界 → Jensen gap が有界 → δ ≈ E[D_KL]（近似的等価性）
- R_2 → ∞ → δ は D_KL の粗い下界にすぎない

### 定理 1.4（XOR-SATでの完全等価性）

XOR-SAT（各制約が x_{i1} ⊕ x_{i2} ⊕ x_{i3} = b_i の形）では：

    D_KL(P_SAT || P_0) = δ    （全インスタンスで正確、期待値ではなく）

**証明**: XOR制約の解集合はアフィン部分空間 {x : Hx = b} （GF(2)上）。
整合的なとき |SAT| = 2^{n - rank(H)}。
各XOR制約は確率 1/2 で充足されるので δ = m · ln 2。
rank(H) = m （m ≤ n で w.h.p.）のとき |SAT| = 2^{n-m}。

    D_KL = ln(2^n / 2^{n-m}) = m · ln 2 = δ    ∎

これはXOR制約が「情報的に完全独立」だからである。各制約が解空間を
正確に半分に分割し、重複がない。

## Direction 2: 制約充足 → 通信路符号化の帰着

### 中心的観察

XOR-SATインスタンスはGF(2)上の線形符号と**同一の数学的対象**である。

    制約集合 {x_{i1} ⊕ x_{i2} ⊕ ... ⊕ x_{ik} = b_i}
    ⟺ パリティ検査行列 H ∈ GF(2)^{m×n}、シンドローム b

### 定理 2.1（XOR-SAT ↔ 線形符号の対応表）

| SAT概念 | 符号理論概念 |
|:--|:--|
| 変数 x_1,...,x_n | 符号語ビット |
| XOR制約 C_i | パリティ検査式 |
| 制約行列 H | パリティ検査行列 |
| 充足解集合 | コセット {x : Hx = b} |
| |SAT| = 2^{n - rank(H)} | 符号のサイズ = 2^{n-m} （符号率 R = 1 - m/n） |
| 充足可能性閾値 α_c = 1 | 符号率 R = 0 の限界 |
| δ = m · ln 2 | 冗長度 (1-R) · n · ln 2 ナット |

### 定理 2.2（閾値 ↔ 通信路容量の対応）

XOR-SATの充足可能性閾値 α_c^{XOR} = 1 は、符号率 R = 0 に対応する：

    R = 1 - m/n = 1 - α

α > 1 で R < 0 → 解なし（容量超過）
α < 1 で R > 0 → 解あり（容量以下）

**存続方程式との接続**:

    S = N_eff · (μ/μ_c) · e^{-δ}

のログ形式 ln S = ln N_eff + ln(μ/μ_c) - δ は：

    ln S = nR · ln 2 + ln(μ/μ_c)    （XOR-SATの場合）

ここで nR · ln 2 = (n - m) · ln 2 = ln|SAT| は符号のエントロピー。

### 定理 2.3（random 3-SAT ↔ 非線形符号のアナロジー）

random 3-SAT は非線形制約による「符号」と見なせる：

| 性質 | XOR-SAT（線形） | random 3-SAT（非線形） |
|:--|:--|:--|
| 1制約あたりの情報量 | ln 2 ≈ 0.693 nat | ln(8/7) ≈ 0.134 nat |
| 閾値 α_c | 1 | ≈ 4.27 |
| 閾値での総情報損失 δ_c | n · ln 2 | 4.27n · ln(8/7) ≈ 0.570n |
| D_KL = δ ? | 完全等価 | δ ≤ E[D_KL]（ギャップあり） |

**なぜ δ_c が異なるか**: XOR-SATでは各制約が ln 2 nat の情報を提供し、
重複なく解空間を分割する。random 3-SATでは各制約が ln(8/7) nat しか
提供せず、かつ制約間の相関で冗長性が生じる。
冗長性の量は第二モーメント比 R_2 で定量化される。

### 定理 2.4（情報効率比 = 減衰率比）

第一モーメント閾値での比：

    α_c^{(1)}(3-SAT) / α_c(XOR-SAT) = ln 2 / ln(8/7) ≈ 5.19

これは**情報効率比**と解釈できる：

    「XOR制約1個の情報量」/「3-SAT制約1個の情報量」= ln 2 / ln(8/7)

XOR制約は3-SAT制約の 5.19倍 情報効率が高い。
したがって同じ総情報損失 δ_c を達成するのに 1/5.19 の制約数で十分。

実験観測値 5.04 ± 0.25 が上界 5.19 を下回るのは、3-SAT制約間の
相関（冗長性）が有効情報量をさらに減少させるため。

## Direction 3: レート歪み理論 → δ_c = 構造的容量

### 中心的主張

存続条件 S > 0 は、Shannon の通信路符号化定理と**構造的に同型**である。

### 定理 3.1（構造的容量定理）

存続方程式 S = N_eff · (μ/μ_c) · e^{-δ} から、存続条件 S ≥ 1 は：

    δ ≤ ln(N_eff · μ/μ_c) ≡ C_struct

ここで C_struct = ln(N_eff · μ/μ_c) を**構造的容量**と定義する。

**Shannon通信路符号化定理との対応**:

| Shannon | 存続方程式 |
|:--|:--|
| 符号語数 2^{nR} | N_eff · μ/μ_c |
| 通信路容量 C | C_struct = ln(N_eff · μ/μ_c) |
| 符号率 R < C で信頼通信可能 | δ < C_struct で存続可能 |
| R > C で誤り確率 → 1 | δ > C_struct で S < 1（崩壊） |
| ノイズ（通信路歪み） | 構造的矛盾（制約蓄積） |

### 定理 3.2（SATでの検証）

random 3-SAT: N_eff = 2^n, μ/μ_c = 1 とすると：

    C_struct = ln(2^n) = n · ln 2

第一モーメント閾値 δ_c^{(1)} = α_c^{(1)} · n · ln(8/7) = (ln 2 / ln(8/7)) · n · ln(8/7) = n · ln 2 ✓

XOR-SAT: δ_c = n · ln 2 = C_struct ✓（完全一致）

### 定理 3.3（逆定理の部分的成立）

Shannon の通信路符号化定理は2つの方向を持つ：
1. **直接定理（achievability）**: R < C → 信頼通信可能
2. **逆定理（converse）**: R > C → 信頼通信不可能

存続方程式の対応：
1. **逆方向（converse）**: δ > C_struct → E[Z] < 1 → Pr[SAT] → 0
   （第一モーメント法＋Markov不等式。**無条件に成立**。）

2. **直接方向（achievability）**: δ < C_struct → Pr[SAT] > 0
   （第二モーメント法＋Paley-Zygmund。**R_2 が有界のとき成立**。）

直接方向の成立範囲：
- XOR-SAT: 完全成立（R_2 = 1、分散なし）
- random 3-SAT: δ < 4.51n · ln(8/7) まで成立（truncated 2nd moment）
- 真の閾値 δ_c ≈ 4.27n · ln(8/7) までの達成には高次モーメント/
  replica symmetry breaking の解析が必要

### 定理 3.4（レート歪み的解釈：最小記述長）

情報源符号化の観点から、δ_c は組織の**最小記述長**と解釈できる：

- 組織の状態空間 Ω（|Ω| = N_eff 個の有効構成）
- 各制約は歪み基準 d(x, x̂) > D_max となる構成を排除
- δ = Σ I(C_i) は制約による**情報的排除量**
- δ_c = ln N_eff は状態空間のエントロピー

δ > ln N_eff のとき、排除した情報量が状態空間のエントロピーを超え、
「記述すべき対象がなくなる」— 存続不可能。

これは**レート歪み関数 R(D) = 0 の条件**に対応する：
歪み許容度 D を超えて情報を失うと、元の信号を復元できない。

## 統合：三角形の閉じ

三方向は独立ではなく、以下の三角形を形成する：

```
        δ = D_KL (Direction 1)
           /            \
          /              \
    制約充足           通信路符号化
    (SAT)              (Channel Coding)
          \              /
           \            /
        C_struct = ln N_eff (Direction 3)
```

Direction 1: δ は KL divergence の下界（独立制約で等価）
Direction 2: 制約はパリティ検査（XOR-SAT = 線形符号）
Direction 3: 存続条件 δ < C_struct は符号化定理 R < C と同型

**閉じた主張**: 存続方程式 S = N_eff · (μ/μ_c) · e^{-δ} は、
Shannon の通信路符号化定理の**有限次元・非線形一般化**と解釈できる。

| 条件 | Shannon | 存続方程式 |
|:--|:--|:--|
| 成立する等式 | R < C ⟺ P_error → 0 | δ < C_struct ⟺ S > 0 |
| 線形の場合 | LDPC/Turbo符号 | XOR-SAT（完全等価） |
| 非線形の場合 | ─ | random 3-SAT（ギャップあり） |
| ギャップの原因 | ─ | replica symmetry breaking |

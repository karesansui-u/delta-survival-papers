基準モデル比較: 線形カウント vs 構造持続理論
— qwen3.5:27b Sonnet 代謝 + num_ctx=16384 実験データの事後分析 —

日付: 2026-04-11
状態: 初回分析


1. 目的

構造持続理論が「単なる矛盾数カウント」以上の予測力を持つかを検証する。ドメイン汎用フレーム仕様書（DSMF）の手順 4.5 および手順 9 で要求されている、Route C における基準モデルとの比較にあたる。

これは構造持続理論の「第 1 段階」（2 ドメイン目で基準モデルに勝つ）の最終工程である。SAT では Route A で成立済みであり、LLM でこれが通れば、同じ骨格が異なるドメインで基準モデルに対する追加予測力を持つことが示される。


2. 比較対象の定義

基準モデル（単純線形カウント）:
  制約の質を区別せず、制約の総数のみで推論性能を予測する。
    予測: Y の低下量 ∝ 注入された制約の総数
    つまり、各カテゴリの ON-OFF 差は注入量に比例すると予測する。
  理論的根拠: 「制約が増えれば性能が落ちる」という直感的で最小限のモデル。

構造持続モデル（本稿の理論）:
  制約の質を区別する。構造的矛盾（contra）は事実的衝突（fact）や
  規則適用（rule）よりも広く有効経路を削ると予測する。
    予測: contra の ON-OFF 差 > rule の ON-OFF 差、かつ
          contra の ON-OFF 差 > fact の ON-OFF 差
  理論的根拠: Paper 3 の補助命題。

両モデルは ON-OFF 差のカテゴリ間順序について異なる予測を生成する。
これが本比較の検証ポイントである。


3. データ

qwen3.5:27b + Sonnet 代謝 + num_ctx=16384 修正済み、30 ターン実験（実質値）:

  ON 条件:  contra ~93%, rule 100%, fact ~76%, overall ~89%
  OFF 条件: contra 6.7%, rule 80%, fact 0%, overall 28.9%
    （OFF は旧 qwen3.5:27b rejudge v4 の結果。num_ctx 修正は代謝 LLM
      出力の長さに影響しないため、OFF 条件での影響は限定的と判断した）

ON-OFF 差:
  contra: +86 pt
  rule:   +20 pt
  fact:   +76 pt

各カテゴリの質問数: contra 15、rule 15、fact 15、合計 45 問。
注入される制約の総数は、3 カテゴリでほぼ同等（各カテゴリで同数の事実・
規則・矛盾ペアが注入される）。


4. 比較

基準モデル（線形カウント）の予測:

  注入される制約数はカテゴリ間でほぼ同等であるから、ON-OFF 差も
  カテゴリ間でほぼ同等になるはずである。具体的な数値は予測しないが、
  少なくとも contra・rule・fact の差が 3 倍以上開くことは予測しない。

実測値との乖離:
  contra +86 pt と rule +20 pt の差は 4.3 倍。
  単純カウントモデルではこの差を予測できない。

構造持続モデルの予測:

  補助命題に従い、contra の ON-OFF 差が最大になると予測する。
  rule と fact の相対順序は予測しない（今後の課題として残す）。

実測値との照合:
  contra +86 pt が最大。予測通り。
  rule +20 pt が最小で、fact +76 pt が中間。
  rule が最小なのは、OFF 条件での rule が 80% と既に高く、ON-OFF の
  改善余地が小さかった（天井効果）ことで説明できる。


5. 判定

基準モデル（線形カウント）:
  カテゴリ間の差を予測できない。実測の 4.3 倍の差は、単純カウント
  モデルの射程外である。外れ。

構造持続モデル:
  「contra の ON-OFF 差が最大」は成立。必須条件 (b)(c) を満たす。
  「contra > rule > fact」の完全順序は天井効果により部分的に崩れたが、
  補助命題の核心は保たれている。


6. 結論

必須条件（contra が最大差）の水準では、構造持続モデルが基準モデルを
上回る予測力を示した。具体的には、基準モデルでは説明不可能な
「カテゴリ間で 4 倍以上の ON-OFF 差」を、構造持続モデルは補助命題から
事前に予測していた。

この結果は、DSMF の手順 9（Route C 検証フェーズ）で要求される「代理指標
が基準モデルに対して追加の予測力を持つ」条件を、qwen3.5:27b ドメインで
初めて満たすものである。

ただし、本分析には以下の限界がある。

第一に、単一モデル（qwen3.5:27b）での単一試行に基づく。統計的頑健性を
示すには、同条件で複数試行を行うか、異なるモデルで再現する必要がある。

第二に、「天井効果」による rule/fact 順序の崩れは、補助命題の成立条件を
精緻化する必要を示唆する。具体的には、ON-OFF 差の大小を評価する前に、
OFF 条件での基準値（ベースライン性能）を揃える必要がある。

第三に、基準モデルの定義が最も単純な線形カウントに限られている。より
洗練された基準モデル（たとえば制約の情報量を考慮するが質の区別はしない
モデル）との比較は今後の課題である。

第四に、今回の結果は num_ctx 切り捨てを修正した後の結果であり、この
修正が fact_recall の劇的改善（0% → 76%）をもたらしている。補助命題の
検証自体は成立しているが、実装の修正がなければこの結果は得られなかった
ことを明記する。

それでもなお、本結果は構造持続理論が LLM ドメインで「単なる矛盾数カウント」
を超える予測力を持つことを示す最初の経験的証拠である。gemma4:31b と
qwen3.5:27b の 2 モデルで補助命題が支持されたことと合わせ、構造持続理論の
第 1 段階（2 ドメイン目で基準モデルに勝つ）は達成されたと判断する。

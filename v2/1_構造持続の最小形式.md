1_構造持続の最小形式
構造持続の最小形式
— 制約の蓄積と情報損失 —

要旨

ある系は、任意の状態で同じ構造を維持できるわけではない。構造の持続は、その構造と両立する状態がなお存在する限られた領域でのみ可能である。本稿は、この事実を表す最小形式を与える。系に対し、その構造を維持できる状態の集合を置き、制約の蓄積をその集合の逐次縮小として表す。各段階の損失を対数比で定義すると、損失は加法的に累積し、残存可能性は指数型で表される。したがって、指数型は追加仮定ではなく、この表現の帰結である。さらに、有効維持資源を導入し、構造持続ポテンシャル
  S = Me^{-L}
を定義する。この形式は、資源の不足と、構造を維持できる状態の減少とを切り分けて記述する。


1. はじめに

本稿の問いは、構造は何によって持続し、何によって失われるのか、である。

構造は、あらゆる状態で同じように維持されるわけではない。ある構造がその構造として持続するためには、それと両立する状態がなお存在していなければならない。問題は、外部から加わる負荷の大きさそのものではない。重要なのは、それによって構造を維持できる状態の領域がどれだけ縮小するかである。

この観点に立つと、構造の持続は、単なる資源残量の問題ではなく、その構造と両立する状態がなお存在するかどうかの問題として捉え直される。したがって、構造の喪失を記述するためには、まず構造を維持できる状態の領域を定め、制約の蓄積がその領域にどのように作用するかを表す必要がある。

本稿の目的は、この事実を表す最小形式を与えることである。ここで中心に置くのは、崩壊そのものではなく、構造が持続可能であるための条件である。以下では、系に対して構造維持可能な状態集合を置き、制約の蓄積をその逐次縮小として表す。そのうえで、各段階の損失を対数比で定義することにより、損失の加法性と残存可能性の指数表現が導かれることを示す。さらに、有効維持資源を導入することで、資源の不足と、構造を維持できる状態の減少という二つの側面を切り分けて記述する。

2. 設定

系 X を固定する。初期時点において、その構造を維持できる状態の集合を
  V^(0)
とする。制約が順に蓄積するとき、構造を維持できる状態の領域は各段階で縮小し、
  V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n)
を得るものとする。

各 V^(i) は、ある可測空間上の可測集合であり、その上に有限測度
  m
を入れる。m(V^(i)) は、その段階でなお残っている構造維持可能性の大きさを表す。m は、個数でも体積でも確率重みでもよい。本稿では、それが同一の系の内部で比較可能であることのみを要請する。

3. 各段階の損失

各段階 i における損失 l_i を
  l_i = -ln( m(V^(i)) / m(V^(i-1)) )
によって定義する。ただし m(V^(i-1)) > 0 とする。もし m(V^(i)) = 0 ならば、
  l_i = +∞
とおく。

l_i は、負荷の大きさそのものではなく、その段階で構造を維持できる状態の集合がどれだけ縮小したかを表す量である。したがって、これは構造維持可能性の対数的損失であり、その意味で情報的損失として読むこともできる。

累積損失 L_n を
  L_n = Σ_{i=1}^n l_i
と定める。

4. 指数表現

命題 1.
  V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n)
を有限測度 m の下での可測集合列とし、
  l_i = -ln( m(V^(i)) / m(V^(i-1)) )
  L_n = Σ_{i=1}^n l_i
と定める。このとき
  m(V^(n)) = m(V^(0)) e^{-L_n}
が成り立つ。

証明.
定義より、
  e^{-l_i} = m(V^(i)) / m(V^(i-1))
であるから、
  e^{-L_n}
    = Π_{i=1}^n e^{-l_i}
    = Π_{i=1}^n m(V^(i)) / m(V^(i-1))
    = m(V^(n)) / m(V^(0)).
よって
  m(V^(n)) = m(V^(0)) e^{-L_n}.
証明終。

重要なのは、この指数型が追加仮定ではなく、状態集合の乗法的縮小を対数的に記述したことの直接の帰結である、という点である。本稿は指数型の経験的妥当性そのものを主張するのではなく、指数型がどこまで表現選択の帰結であり、どこから個別領域の経験的問題になるかを分離することを目的とする。


5. 構造持続ポテンシャル

命題1は、構造を維持できる状態の集合がどのように縮小するかを与える。しかし、構造の持続は、状態の広がりだけで決まるとは限らない。実際の系では、その構造を保つために利用できる資源もまた重要である。そこで、命題1の幾何学的縮小とは別に、有効維持資源を
  M > 0
として導入し、構造持続ポテンシャル S を
  S = Me^{-L_n}
と定める。これは命題1の直接の帰結ではなく、資源要因を組み込んだ拡張的定義である。

ここで M は、系がその構造を維持するために利用できる有効資源の大きさを表す。個別の系において、それはエネルギー、時間、冗長性、回復可能性、計算余地、あるいは流れを維持する能力として具体化されうる。本稿では、その内訳には立ち入らない。

構造の持続は、ある閾値 S_c に対して
  S < S_c
となるとき失われる。このとき観測される現象は、崩壊、機能不全、相転移、自己同一性の喪失など、系に応じて異なる形をとる。

6. この形式が含意すること

6.1 制約の質は量に優先しうる

各段階の寄与は、制約の数ではなく、状態集合の縮小率によって測られる。したがって、少数の制約でも、構造を維持できる状態を大きく減少させるならば支配的になりうる。逆に、多数の制約でも、その縮小率が小さければ、累積損失は大きくならない。

6.2 資源と損失は異なる側面をなす

S = M exp(-L_n) において、M と L_n は構造持続の異なる側面を表す。一方は維持資源の大きさであり、他方は構造を維持できる状態の縮小である。いずれか一方が十分に悪化すれば、構造持続ポテンシャルは失われる。したがって、資源が残っていても構造的に維持不能な場合があり、逆に構造を維持できる状態がなお残っていても、資源不足によって維持不能になる場合がある。

6.3 臨界領域が現れうる

S が閾値 S_c に近い領域では、小さな追加損失が構造の持続を失わせうる。したがって、系によっては、緩やかな劣化ではなく、急峻な転移に近い挙動が現れうる。

7. 例

この形式は、個別の系に応じて異なる対象に対応づけられる。

SAT では、V^(i) は、その段階でなお残っている充足割当ての集合に対応する。各制約はこの集合を縮小させ、累積損失は解空間の収縮として読まれる。

個別領域において制約型ごとの損失が評価可能な場合には、本形式は比率予測へ接続しうる。SAT はその具体例である。詳細は別稿に譲る。

推論系では、V^(i) は、整合性を保ったまま推論を完了できる状態または経路の集合に対応する。不整合や未整理の上書きは、その集合を縮小させる制約として現れる。

散逸構造や生物系では、V^(i) は、境界維持、流れ、代謝、温度勾配、化学ポテンシャル差などの条件と両立する状態の集合に対応する。このとき構造持続は、静止した秩序としてではなく、流れの中で維持される自己同一性として現れる。

本稿の主張は、これらを同一視することにはない。主張は、それらがいずれも「構造を維持できる状態集合の縮小」という同じ最小形式のもとで記述可能である、という一点にある。

8. 限界

本稿は、すべての系に対して一つの測度 m、一つの資源量 M、一つの閾値 S_c を与えるものではない。また、個別の系において L_n が一意に測定できることも主張しない。さらに、制約の追加が常にあらゆる表現において縮小方向にのみ働くことを、一般定理として主張するものでもない。

とくに、本形式は制約蓄積が支配的な局面を記述するものであり、学習・修復・冗長化など、構造を維持できる状態集合を再拡大する過程は明示的に含まれていない。したがって、本形式は、散逸構造を含む広い範囲の系に対して持続条件の骨格を与えうるが、流れによる自己維持や再編成のような動的過程そのものを記述するものではない。そのような動的過程を含む理論は、今後の拡張として別途扱う必要がある。

測度 m の具体化は個別領域に依存する。SAT では充足割当ての数え上げとして自然に定まるが、推論系のように直接の測定が困難な領域では、観測指標を介した間接的な接近が必要になる。このとき本形式は、指標の探索と検証のための枠組みとして機能する。すなわち、本形式が与える順序予測や方向予測に合致する指標は妥当な代理指標として残り、合致しない指標は棄却される。したがって、測度の直接測定が困難であることは、本形式の適用を不可能にするのではなく、代理指標の発見を構造化された経験的課題として位置づける。

本稿が与えるのは、より限定された形式である。すなわち、ある系について構造を維持できる状態集合が定まり、制約の蓄積をその集合の縮小として表せるならば、損失は対数比として定義でき、残存可能性は指数型で表される、ということである。

9. 結論

構造の持続は、単なる資源残量の問題ではない。構造維持条件と両立する状態がなお存在するかどうかの問題でもある。本稿では、その事実を表す最小形式を与えた。構造を維持できる状態集合の逐次縮小を対数比で表すと、損失は加法的に累積し、残存可能性は指数型で表される。指数型は仮定ではなく、この表現の帰結である。さらに、有効維持資源を導入することで、構造持続ポテンシャル
  S = Me^{-L}
を得た。この形式は、資源の不足と、構造を維持できる状態の減少とを切り分けつつ、構造持続の喪失を一つの形で記述する。

本稿の役割は、この形式をできる限り薄く与えることにある。個別領域における測度の選択、資源量の定義、閾値の同定、実証、および応用は別稿に譲る。

付記

本文では簡潔さのために L_n と書いたが、文脈が明らかなときは単に L と書いてよい。したがって、中心式は
  S = Me^{-L}
である。

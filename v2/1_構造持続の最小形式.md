1_構造持続の最小形式
構造持続の最小形式
— 制約蓄積による構造損失 —

要旨

構造が失われるのは、資源が尽きるときだけではない。資源がなお残っていても、制約の蓄積によって構造を維持できる状態の領域が縮小し、ついにはその構造を保てる状態が残らなくなることがある。本稿は、この事実を表す最小形式を与える。

系に対し、その構造を維持できる状態の集合を置き、制約の蓄積をその集合の逐次縮小として表す。段階損失に対する自然な公理系（比率依存性、正規化、加法性、連続性）のもとで、損失の関数形は対数比として一意に強制される。そのうえで、累積損失の望遠鏡積として残存可能性の指数表現が恒等的に成り立つ。したがって、指数型は追加仮定ではなく、損失測度の公理系から二段階で導かれる必然的な形式である。さらに、有効維持資源を導入して構造持続ポテンシャル
  S = Me^{-L}
を定義する。この形式は、資源不足と構造的維持可能性の減少とを切り分けて記述する。


1. はじめに

本稿の問いは、構造は何によって持続し、何によって失われるのか、というものである。
構造の喪失を資源の減少だけで捉えるなら、この問いには十分に答えられない。
むしろ問うべきなのは、構造を維持できる状態の領域が何によって縮小するのか、である。

本稿の目的は、この事実を表す最小形式を与えることである。以下では、系に対して構造維持可能な状態集合を置き、制約の蓄積をその逐次縮小として表す。段階損失に対する自然な公理系から損失の関数形が対数比として一意に強制されることを示し、そのうえで累積損失の望遠鏡積から残存可能性の指数表現を導く。さらに、有効維持資源を導入することで、資源の不足と構造維持可能性の減少という二つの側面を切り分けて記述する。

2. 設定

系 X を固定する。本稿でいう「構造」とは、一般的な形そのものではなく、その系で維持を問題にする関係・機能・同一性の総称であり、その具体的内容は個別領域に委ねる。初期時点において、その構造を維持できる状態の集合を
  V^(0)
とする。制約が順に蓄積するとき、構造を維持できる状態の領域は各段階で縮小し、
  V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n)
を得るものとする。本稿は、制約追加が支配的であり、学習や修復による再拡大を無視できる局面を対象とする。

ここで区別したいのは、少なくとも三つの層である。第一に、物質・パラメータ・計算基盤などとしての基体の存在。第二に、「この人格」「この知識体系」「この推論系」のような、その構造としての同一性。第三に、その構造が担う機能である。本稿が主に扱うのは第二と第三であり、制約蓄積によって問題になるのは、基体そのものの消滅ではなく、ある構造として存在し続けること、およびその構造が担う機能を維持できることの失敗である。したがって、ここでいう構造の喪失は、多くの場合、無への消滅ではなく、別の構造への遷移、構造的再編、相転移として観測されうる。

各 V^(i) は、ある可測空間上の可測集合であり、その上に固定された有限測度
  m
を入れる。m(V^(i)) は、その段階でなお残っている構造維持可能性の大きさを表す。m は個数でも体積でも確率重みでもよいが、本稿では、同一の系の同一表現の上で段階を通じて比較可能な一つの測度であることのみを要請する。なお、m(V^(i)) は系が現に占めている状態ではなく、その段階でなお構造維持と両立しうる状態の余地を表す。

3. 制約ごとの構造損失

各段階の損失を、縮小前後の集合対からの写像として与える。以下では、この損失の関数形が自然な公理系から一意に強制されることを示し、そのうえで段階損失 l_i と累積損失 L_n を定義する。

3.1 損失測度の公理

本稿の理論の主対象は、残存比率 r ∈ (0, 1] に対する損失尺度
  f : (0, 1] → [0, ∞)
である。縮小前後の集合対 A ⊇ B (m(A) > 0) に対して定まる損失 D(A, B) ≥ 0 は、この f を用いた集合論的表現
  D(A, B) = f( m(B) / m(A) )
として読む。したがって公理は f に対して立て、D はその表現として位置づける。この順序は、Shannon (1948) がエントロピーを特定の有限標本空間上の関数ではなく確率分布上の関数として公理化したことに対応する。以下では A, B は §2 の V^(i-1), V^(i) に対応するものとし、f に対して次の公理を置く。

公理 B1 (比率依存性・表現)
縮小前後の集合対 A ⊇ B における損失は、f と残存比率 m(B)/m(A) によって
  D(A, B) = f( m(B) / m(A) )
と表される。すなわち、D は残存比率のみに依存し、測度の絶対量には依存しない。f は比率空間 (0, 1] 全体で定義される関数であり、特定のドメインで実現される比率の部分集合だけに限定されない。

公理 B2 (正規化)
D(A, A) = 0、すなわち f(1) = 0。縮小が起きていなければ損失はない。

公理 B3 (加法性)
f は関数方程式
  f(r_1 r_2) = f(r_1) + f(r_2)   (r_1, r_2 ∈ (0, 1])
を満たす。

公理 B4 (連続性)
f は (0, 1] 上で連続。

公理 B5 (単調性)
縮小が大きいほど損失は大きい。すなわち f は (0, 1] 上で単調非増加。

B1 は、構造持続で問題とするのが残存量の絶対値ではなく比率であるという立場を反映する。

B3 は、f に対する関数方程式として直接定式化している。これは損失尺度に対する二つの自然な要請の関数形としての表現であり、どちらも同じ関数方程式に帰着する。

第一は連鎖加法性（集合論的形式）である。A ⊇ B ⊇ C のとき
  D(A, C) = D(A, B) + D(B, C)
が成り立つべきだ、という要請であり、縮小列を途中点で二分割したときに全体の損失が二部分の損失の和に一致すべきだ、という累積損失そのものを成立させるための要請である。

第二は示量性（独立分解形式）である。状態空間が二つの独立なサブシステムの積に分解され、それぞれに独立に制約が作用するとき、全体系の損失は部分系の損失の和であるべきだ、という要請である。統計力学におけるエントロピーや自由エネルギーの示量性、および Hartley (1928) / Shannon (1948) による情報量の加法性と同じ骨格を持つ。

B3 は比率空間 (0, 1] 上の損失関数 f に対する公理であり、各ドメインはその全域ではなく、観測可能な比率の部分集合のみを実現する。したがって、原子的な測度空間で未実現の比率が残ることは理論の限界ではなく、比率尺度上で定義された損失関数を各ドメインが部分的にサンプリングすることの反映である。この見方では、集合論的な実現可能性は表現問題であって、B3 の本質ではない。Route A のように実現比率が十分豊かなドメイン（例: SAT における連続節密度 α）では、B3 はほぼそのまま検証可能である。Route C のような高次元ドメインでは、f 全域の厳密実現ではなく、観測可能部分での予測力として評価される（別稿の Route 分類を参照）。連鎖加法性と示量性は、いずれも B3 の関数方程式に対する自然な動機づけを与える集合論的・空間構造的形式にあたる。

ここで強調すべきは、B3 が確率的独立性とは別物であることである。B3 は f の関数方程式であり、その背後にある連鎖加法性も示量性もいずれも集合・空間構造に関する要請であって、制約の生成過程を確率的にモデル化する際の独立性（別稿の A3）とは別層の概念である。有限測度 m の乗法性
  m(C)/m(A) = ( m(B)/m(A) ) × ( m(C)/m(B) )
は測度の定義から自動的に成り立つため、B3 の背後にある集合論的形式は統計的仮定を一切必要としない。

補助的な解釈として、各段階損失 l_i は条件付き残存確率 p_i = m(V^(i))/m(V^(i-1)) に対する self-information
  -log p_i
としても読める。これは Shannon (1948) の情報量定義との局所的な対応を与えるものであり、集合論的な f の大域的拡張を経由せずに l_i を段階ごとに解釈できる。ただし、本稿の主証明は §3.2 の公理系 B1–B4 からの Cauchy 関数方程式によって独立に閉じており、情報理論の公理系そのものを主証明の前提としない。self-information 解釈は、本稿の公理化と情報理論の公理化とが同じ結論に収束することを示す補助線として位置づけられる。

3.2 対数比の一意性

定理 (対数比の一意性).
B1–B4 を満たす f は、ある k ≥ 0 に対して
  f(r) = -k ln r
と一意に表される。f が恒等的にゼロでなければ k > 0 であり、このとき B5 も自動的に満たされる。

証明.
B3 より、任意の r_1, r_2 ∈ (0, 1] に対して
  f(r_1 r_2) = f(r_1) + f(r_2)
が成り立つ。次に t ≥ 0 に対して g を
  g(t) := f(e^{-t})
で定める。すると上式は
  g(s + t) = g(s) + g(t)
  (s, t ≥ 0)
となり、B4 により g は連続である。連続な加法的関数は Cauchy の関数方程式の一意解として
  g(t) = kt
  (k := g(1))
の形を取る。f ≥ 0 から k ≥ 0。したがって
  f(r) = g(-ln r) = -k ln r。
証明終。

この定理により、B1–B4 を満たす損失関数は、単位尺度 k > 0 の選択を除いて一意に対数比として表される。本稿では k = 1 と取り、これを構造ナット (structural nat) の単位規約と呼ぶ。

3.3 段階損失と累積損失

k = 1 の単位規約のもとで、各段階 i における損失は
  l_i = D(V^(i-1), V^(i)) = -ln( m(V^(i)) / m(V^(i-1)) )
と書かれる。m(V^(i-1)) > 0 を仮定し、m(V^(i)) = 0 のときは
  l_i = +∞
とおく。

l_i は、負荷の大きさそのものではなく、その段階で構造を維持できる状態の集合がどれだけ縮小したかを表す量である。言い換えれば、構造を維持しうる可能性の量が段階ごとに削られていくことを表す対数的損失であり、その意味で、構造維持可能性に関する自由度の喪失として読むことができる。

累積損失 L_n を
  L_n = Σ_{i=1}^n l_i
と定める。B1 と B3 から、縮小列 V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n) に対して
  D(V^(0), V^(n)) = f( m(V^(n)) / m(V^(0)) )
                 = f( ∏_{i=1}^n ( m(V^(i)) / m(V^(i-1)) ) )
                 = Σ_{i=1}^n f( m(V^(i)) / m(V^(i-1)) )
                 = Σ_{i=1}^n l_i
                 = L_n
が代数的に従う。すなわち累積損失は、始点と終点の比率に対する f の値に一致する。

3.4 注意

第一に、B3 の厳密加法性を q-加法性
  f(rs) = f(r) + f(s) - (1 - q) f(r) f(s)
に緩めれば、損失尺度 f として q-対数
  ln_q(x) = (x^{1-q} - 1) / (1 - q)
に対応する
  f(r) = -ln_q(r)
が導出され、§4 の指数表現は q-指数に置き換わる（Tsallis 型の拡張）。ここで q-対数が満たす加法則は
  ln_q(xy) = ln_q(x) + ln_q(y) + (1 - q) ln_q(x) ln_q(y)
であり、f = -ln_q を通じて書き換えると上の符号になる。本稿の立場は、厳密加法性こそが累積損失という概念を最も自然に支える要請である、というものである。q-加法的拡張は別系統の理論として位置づけ、本稿の対象外とする。

第二に、本節の結果は「測度 m が与えられたときに、損失の自然な関数形は何か」を一意に定める定理であって、m そのものの選び方までを一意化するものではない。各ドメインにおける m の具体化は個別の経験的課題として残る（§8）。

第三に、B4 を可測性に緩めてもよい。連続加法関数の特徴づけ（Cauchy）は可測加法関数の特徴づけ（Steinhaus ほか）としても同じ結論を与える。病的な非可測解を避ける最小の正則性条件として、連続性または可測性のいずれかがあればよい。

4. 残存可能性の指数表現

命題 1.
  V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n)
を有限測度 m の下での可測集合列とし、
  l_i = -ln( m(V^(i)) / m(V^(i-1)) )
  L_n = Σ_{i=1}^n l_i
と定める。このとき
  m(V^(n)) = m(V^(0)) e^{-L_n}
が成り立つ。

証明.
定義より、
  e^{-l_i} = m(V^(i)) / m(V^(i-1))
であるから、
  e^{-L_n}
    = Π_{i=1}^n e^{-l_i}
    = Π_{i=1}^n m(V^(i)) / m(V^(i-1))
    = m(V^(n)) / m(V^(0)).
よって
  m(V^(n)) = m(V^(0)) e^{-L_n}.
証明終。

したがって指数型は追加仮定ではなく、§3 の公理系 B1–B4 から強制された対数比損失の望遠鏡積として恒等的に成り立つ。言い換えれば、指数型は損失測度の公理系と測度の乗法性から二段階で導かれる必然的な形式である。指数型それ自体は経験法則ではなく、経験的内容は V^(i) と m の具体化、および各制約に対応する縮小率の差異に属する。

本形式が単なる記述以上の予測力を持つのは、制約ごとの縮小率 l_i が異なりうることを通じて、制約の質が量に優先しうるという順序予測を生むときである。この予測が個別ドメインで検証可能かどうかが、本形式の経験的試練となる。

対数比で定義された段階損失の加法的累積から指数型が現れることは、既知の数学的構造に属する。望遠鏡積による指数表現は、生存解析のハザード累積、統計力学の分配関数、情報理論の容量境界と形式的に同型であり、指数型そのものに数学的新規性はない。§3 の対数比の一意性定理もまた、Hartley (1928) および Shannon (1948) が情報量を加法性公理から特徴づけた論法（Cauchy の関数方程式の連続解）を、構造維持可能性の縮小率という異なる対象に適用したものである。したがって、本節の寄与は公理化論法そのものの新規性ではなく、その論法を構造持続の設定で機能させ、指数型を損失測度の公理系から導く点にある。

本稿の寄与は、この骨格を知能的・推論的・学習的な構造の崩壊へ概念写像することにある。経験的内容は、指数型そのものではなく、各ドメインにおいて何を構造維持可能状態とみなし、どの測度で比較し、どの制約がどれだけの縮小率を持つかという具体化に属する。ここでいう基準モデルとは、制約の個数や文脈の長さなど、質を区別しない単純な数え上げによる予測を指す。本稿が問うのは、この写像手順が、異なるドメインにおいて基準モデルに対する追加の予測力を生むかどうかである。具体化と検証は別稿に委ねる。


5. 構造持続ポテンシャル

命題1は、構造を維持できる状態集合の残存量がどのように表されるかを与える。しかし、構造の持続は、その残存量だけで決まるとは限らない。実際の系では、その構造を保つために利用できる資源もまた重要である。そこで、資源要因と構造維持可能性の縮小要因とを切り分けて表す最小の定式化として、命題1で与えた指数表現とは別に、有効維持資源を
  M > 0
として導入し、構造持続ポテンシャル S を
  S = Me^{-L_n}
と定める。これは命題1の直接の帰結ではなく、資源要因を組み込んだ拡張的定義である。

M は m(V^(0)) と同一視される必要はない。m(V^(0)) は構造維持可能状態の初期的広がりであるのに対し、M は系がその構造を維持するために利用できる有効資源の大きさを表す。個別の系において、それはエネルギー、時間、冗長性、回復可能性、計算余地、あるいは流れを維持する能力として具体化されうる。本稿では、その内訳には立ち入らない。

最も単純には、構造の持続はある閾値 S_c に対して
  S < S_c
となるとき失われる。このとき観測される現象は、崩壊、機能不全、相転移、自己同一性の喪失など、系に応じて異なる形をとる。

ここで失われるのは、基体の存在そのものではなく、その構造としての同一性や機能である。したがって、閾値を下回った系は、必ずしも消滅するとは限らず、別の構造として存続することがある。

6. 含意

6.1 制約の質は量に優先しうる

各段階の寄与は、制約の数ではなく、状態集合の縮小率によって測られる。したがって、少数の制約でも、構造を維持できる状態を大きく減少させるならば支配的になりうる。逆に、多数の制約でも、その縮小率が小さければ、累積損失は大きくならない。したがって本形式は、制約の個数よりも、どの制約がより大きく維持可能状態集合を削るかに関する順序予測を与える。

6.2 資源と損失は異なる側面をなす

S = Me^{-L_n} において、M と L_n は構造持続の異なる側面を表す。一方は維持資源の大きさであり、他方は構造を維持できる状態の縮小である。いずれか一方が十分に悪化すれば、構造持続ポテンシャルは失われる。したがって、資源が残っていても構造を維持できる状態が失われる場合があり、逆に構造を維持できる状態がなお残っていても、資源不足によって維持が失われる場合がある。

6.3 臨界領域が現れうる

S が閾値 S_c に近い領域では、小さな追加損失でも構造の持続が失われうる。したがって、閾値的な挙動を示す系では、緩やかな劣化ではなく、急峻な転移に近い挙動が現れうる。

7. 適用例：推論系・SAT・散逸構造

この形式は、個別の系に応じて異なる対象に対応づけられる。

推論系では、V^(i) は、整合性を保ったまま推論を完了できる状態または経路の集合に対応する。不整合や未整理の上書きは、その集合を縮小させる制約として現れる。

SAT では、V^(i) は、その段階でなお残っている充足割当ての集合に対応する。各制約はこの集合を縮小させ、累積損失は解空間の収縮として読まれる。制約型ごとの損失が評価可能な場合には、本形式は比率予測へ接続しうる。SAT はその具体例の一つである。詳細は別稿に譲る。

散逸構造や生物系では、V^(i) は、境界維持、流れ、代謝、温度勾配、化学ポテンシャル差などの条件と両立する状態の集合に対応する。このとき構造持続は、静止した秩序としてではなく、流れの中で維持される自己同一性として現れる。

本稿の主張は、これらを同一視することにはない。主張は、それらがいずれも「構造を維持できる状態集合の縮小」という同じ最小形式のもとで記述可能である、という一点にある。

8. 限界

本稿は、すべての系に対して一つの測度 m、一つの資源量 M、一つの閾値 S_c を与えるものではない。また、個別の系において L_n が一意に測定できることも主張しない。さらに、制約の追加が常にあらゆる表現において縮小方向にのみ働くことを、一般定理として主張するものでもない。

とくに、本形式は制約蓄積が支配的な局面を記述するものであり、学習・修復・冗長化など、構造を維持できる状態集合を再拡大する過程は明示的に含まれていない。したがって、本形式は、散逸構造を含む広い範囲の系に対して持続条件の骨格を与えうるが、流れによる自己維持や再編成のような動的過程そのものを記述するものではない。そのような動的過程を含む理論は、今後の拡張として別途扱う必要がある。

測度 m の具体化は個別領域に依存する。SAT では充足割当ての数え上げとして自然に定まるが、推論系のように直接の測定が困難な領域では、観測指標を介した間接的な接近が必要になる。このとき本形式は、指標の探索と検証のための枠組みとして機能する。すなわち、本形式が与える順序予測や方向予測に合致する指標は妥当な代理指標として残り、合致しない指標は棄却される。したがって、測度の直接測定が困難であっても、本形式の適用が不可能になるわけではない。問題は、代理指標の発見を構造化された経験的課題として引き受けることである。

本稿が与えるのは、構造持続を記述するための条件つきの最小形式である。すなわち、ある系について構造を維持できる状態集合が定まり、制約の蓄積をその集合の縮小として表せるならば、損失は対数比として定義でき、残存可能性は指数型で表される、ということである。

9. 結論

構造の持続は、単なる資源残量の問題ではない。構造維持条件と両立する状態がなお存在するかどうかの問題でもある。本稿では、その事実を表す最小形式を与えた。構造を維持できる状態集合の逐次縮小に対して、損失測度に関する自然な公理系（比率依存性、正規化、加法性、連続性）が対数比の関数形を一意に強制し、その望遠鏡積として残存可能性の指数表現が恒等式として従う。指数型は仮定ではなく、公理系と測度の乗法性から二段階で導かれる帰結である。さらに、有効維持資源を導入することで、構造持続ポテンシャル
  S = Me^{-L}
を得た。この形式は、資源の不足と構造を維持できる状態の減少とを切り分けつつ、基体の存在そのものではなく、その構造としての持続の喪失を一つの形式で記述する。

本稿が与えるのは、個別理論に先立つ最小の骨格である。測度の選択、資源量の定義、閾値の同定、実証、および応用は別稿に譲る。

付記

本文では簡潔さのために L_n と書いたが、文脈が明らかなときは単に L と書いてよい。したがって、中心式は
  S = Me^{-L}
である。

1_構造持続の最小形式
構造持続の最小形式
— 制約蓄積による構造損失 —

要旨

構造が失われるのは、資源が尽きるときだけではない。資源がなお残っていても、制約の蓄積によって構造を維持できる状態の領域が縮小し、ついにはその構造を保てる状態が残らなくなることがある。本稿は、この事実を表す最小形式を与える。

系に対し、その構造を維持できる状態の集合を置き、制約の蓄積をその集合の逐次縮小として表す。段階損失に対する自然な公理系（比率依存性、正規化、加法性、連続性）のもとで、損失の関数形は対数比として一意に強制される。そのうえで、累積損失の望遠鏡積として残存可能性の指数表現が恒等的に成り立つ。したがって、指数型は追加仮定ではなく、損失測度の公理系から二段階で導かれる必然的な形式である。さらに、有効維持資源を導入して構造持続ポテンシャル
  S = Me^{-L}
を定義する。この形式は、資源不足と構造的維持可能性の減少とを切り分けて記述する。


1. はじめに

本稿の問いは、構造は何によって持続し、何によって失われるのか、というものである。
構造の喪失を資源の減少だけで捉えるなら、この問いには十分に答えられない。
むしろ問うべきなのは、構造を維持できる状態の領域が何によって縮小するのか、である。

本稿の目的は、この事実を表す最小形式を与えることである。以下では、系に対して構造維持可能な状態集合を置き、制約の蓄積をその逐次縮小として表す。段階損失に対する自然な公理系から損失の関数形が対数比として一意に強制されることを示し、そのうえで累積損失の望遠鏡積から残存可能性の指数表現を導く。さらに、有効維持資源を導入することで、資源の不足と構造維持可能性の減少という二つの側面を切り分けて記述する。

2. 設定

系 X を固定する。本稿でいう「構造」とは、一般的な形そのものではなく、その系で維持を問題にする関係・機能・同一性の総称であり、その具体的内容は個別領域に委ねる。初期時点において、その構造を維持できる状態の集合を
  V^(0)
とする。一般形では、制約が順に蓄積するとき、構造を維持できる状態の領域は各段階で縮小し、
  V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n)
を得るものとする。

命題（制約追加モデルにおける縮小性）。
各段階で新条件 C_i が既存の構造維持条件に追加され、その段階で構造を維持できる状態集合が
  V^(i) = V^(i-1) ∩ C_i
と表されるなら、
  V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n)
が成り立つ。

証明。
各 i について
  V^(i) = V^(i-1) ∩ C_i ⊆ V^(i-1)
であるから、帰納的に主張が従う。証明終。

この命題は、Paper 2 で A1 として用いる縮小条件に対する一つの自然な十分条件を与える。したがって A1 型の縮小性は、宇宙的一般法則としてではなく、制約追加レジームでは命題として導かれる。これに対し、学習・修復・冗長化・外部支援・ロールバックのように、構造を維持できる状態集合を再拡大しうる作用が支配的な局面では、一般に
  V^(i) = R_i(V^(i-1))
のような拡大型作用が入りうるため、上の縮小列は自動的には従わない。したがって本稿は、制約追加が支配的であり、これらの再拡大効果を無視できる局面、すなわち収縮モードを主対象とする。

ここで問題にしているのは、基体そのものの消滅ではなく、ある構造として存在し続けること、およびその構造が担う機能を維持できることの失敗である。したがって、本稿でいう構造の喪失は、多くの場合、無への消滅ではなく、別の構造への遷移、構造的再編、相転移として観測されうる。

各 V^(i) は、ある可測空間上の可測集合であり、その上に固定された有限測度
  m
を入れる。m(V^(i)) は、その段階でなお残っている構造維持可能性の大きさを表す。m は個数でも体積でも確率重みでもよいが、本稿では、同一の系の同一表現の上で段階を通じて比較可能な一つの測度であることのみを要請する。なお、m(V^(i)) は系が現に占めている状態ではなく、その段階でなお構造維持と両立しうる状態の余地を表す。

この最小理論は、事前固定された構造維持問題に対する表現定理として読むべきである。ここでいう構造維持問題とは、系 X、初期構造を維持できる状態集合 V^(0)、比較に用いる有限測度 m、段階列 {C_i}、および時間地平 T が観測前に外部から固定された設定をいう。したがって、観測後に別の構造 V'^(0) や別の測度 m' を選び直してよいなら、その再記述は新しい問題設定として扱われ、本稿の元の予測の成否を遡って救済するものではない。

適用可能性条件.
P1 (事前固定性): V^(0) と m は観測前に固定される。結果を見てから、より都合のよい構造や測度に選び直してはならない。
P2 (段階列と地平の事前固定): 制約列 {C_i} と時間地平 T は観測前に固定される。崩壊や遷移を見てから区切りを引き直してはならない。
P3 (遷移のデータ化): V^(0) の持続が破れた事実は、それ自体をデータ点として記録する。別の構造 V'^(0) のもとでの再解析は、新しい問題設定または新しい実験として扱う。
P4 (非自明性): 初期構造領域は空でも全空間でもない。少なくとも 0 < m(V^(0)) < m(X) を要請し、自明な singleton や tautology に近い表現で理論を空虚化しない。
P5 (表現安定性): 対象構造を保つ自然な coarse-graining や再記述に対して、L や S の予測が本質的に変わってはならない。もし予測が表現の取り方だけで大きく変わるなら、その差は現象ではなく表現の産物である。

命題（事後的表現選択による空虚化）。
任意の有限単調減少列
  s_0 ≥ s_1 ≥ ... ≥ s_n > 0
に対し、ある可測空間 X、有限測度 m、縮小列
  V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n)
が存在して
  m(V^(i)) = s_i
を満たす。

証明。
たとえば X = [0, s_0]、m を Lebesgue 測度、
  V^(i) = [0, s_i]
とおけばよい。このとき各 V^(i) は可測であり、s_i の単調減少性から
  V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n)
が成り立ち、また
  m(V^(i)) = s_i
である。証明終。

したがって、V^(0) や m を観測後に十分自由に選んでよいなら、任意の有限観測系列に適合する縮小表現を構成でき、本理論は経験的内容を失う。上の P1–P5 は、この空虚化を防ぎ、本稿の最小形式を反証可能な理論として読むための適用可能性条件である。

3. 制約ごとの構造損失

各段階の損失を、縮小前後の集合対からの写像として与える。以下では、この損失の関数形が自然な公理系から一意に強制されることを示し、そのうえで段階損失 l_i と累積損失 L_n を定義する。

3.1 損失測度の公理

本稿の理論の主対象は、残存比率 r ∈ (0, 1] に対する損失尺度
  f : (0, 1] → [0, ∞)
である。縮小前後の集合対 A ⊇ B (m(A) > 0) に対して定まる損失 D(A, B) ≥ 0 は、この f を用いた集合論的表現
  D(A, B) = f( m(B) / m(A) )
として読む。したがって公理は f に対して立て、D はその表現として位置づける。この順序は、Shannon (1948) がエントロピーを特定の有限標本空間上の関数ではなく確率分布上の関数として公理化したことに対応する。以下では A, B は §2 の V^(i-1), V^(i) に対応するものとし、f に対して次の公理を置く。

公理 B1 (比率依存性・表現)
縮小前後の集合対 A ⊇ B における損失は、f と残存比率 m(B)/m(A) によって
  D(A, B) = f( m(B) / m(A) )
と表される。すなわち、D は残存比率のみに依存し、測度の絶対量には依存しない。f は比率空間 (0, 1] 全体で定義される関数であり、特定のドメインで実現される比率の部分集合だけに限定されない。

公理 B2 (正規化)
D(A, A) = 0、すなわち f(1) = 0。縮小が起きていなければ損失はない。

公理 B3 (加法性)
f は関数方程式
  f(r_1 r_2) = f(r_1) + f(r_2)   (r_1, r_2 ∈ (0, 1])
を満たす。

公理 B4 (連続性)
f は (0, 1] 上で連続。

補助条件 B5 (単調性)
縮小が大きいほど損失は大きい。すなわち f は (0, 1] 上で単調非増加。これは B1–B4 から従う性質であるが、損失尺度の直観的意味を先に見やすくするため、ここでは補助条件として併記しておく。

B1 は、構造持続で問題とするのが残存量の絶対値ではなく比率であるという立場を反映する。

B3 は f に対する関数方程式として直接定式化している。その自然な動機づけは、縮小列を分割したとき全体損失が部分損失の和に一致すべきだという連鎖加法性と、独立サブシステムでは損失が足し合わさるべきだという示量性にある。この骨格は、Hartley (1928) / Shannon (1948) による情報量の加法性公理化と同型である。本稿では詳論しないが、B3 は損失尺度 f の関数形に関する要請であって、制約生成過程の確率的独立性とは別物である。

3.2 対数比の一意性

定理 (対数比の一意性).
B1–B4 を満たす f は、ある k ≥ 0 に対して
  f(r) = -k ln r
と一意に表される。f が恒等的にゼロでなければ k > 0 であり、このとき上の B5 も自動的に満たされる。

証明.
B3 より、任意の r_1, r_2 ∈ (0, 1] に対して
  f(r_1 r_2) = f(r_1) + f(r_2)
が成り立つ。次に t ≥ 0 に対して g を
  g(t) := f(e^{-t})
で定める。すると上式は
  g(s + t) = g(s) + g(t)
  (s, t ≥ 0)
となり、B4 により g は連続である。連続な加法的関数は Cauchy の関数方程式の一意解として
  g(t) = kt
  (k := g(1))
の形を取る。f ≥ 0 から k ≥ 0。したがって
  f(r) = g(-ln r) = -k ln r。
証明終。

この定理により、B1–B4 を満たす損失関数は、単位尺度 k > 0 の選択を除いて一意に対数比として表される。本稿では k = 1 と取り、これを構造ナット (structural nat) の単位規約と呼ぶ。

3.3 段階損失と累積損失

k = 1 の単位規約のもとで、各段階 i における損失は
  l_i = D(V^(i-1), V^(i)) = -ln( m(V^(i)) / m(V^(i-1)) )
と書かれる。m(V^(i-1)) > 0 を仮定し、m(V^(i)) = 0 のときは
  l_i = +∞
とおく。

l_i は、負荷の大きさそのものではなく、その段階で構造を維持できる状態の集合がどれだけ縮小したかを表す量である。言い換えれば、構造を維持しうる可能性の量が段階ごとに削られていくことを表す対数的損失であり、その意味で、構造維持可能性に関する自由度の喪失として読むことができる。

累積損失 L_n を
  L_n = Σ_{i=1}^n l_i
と定める。B1 と B3 から、縮小列 V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n) に対して
  D(V^(0), V^(n)) = f( m(V^(n)) / m(V^(0)) )
                 = f( ∏_{i=1}^n ( m(V^(i)) / m(V^(i-1)) ) )
                 = Σ_{i=1}^n f( m(V^(i)) / m(V^(i-1)) )
                 = Σ_{i=1}^n l_i
                 = L_n
が代数的に従う。すなわち累積損失は、始点と終点の比率に対する f の値に一致する。

3.4 備考

本節の結果は、測度 m が与えられたときに損失の自然な関数形を一意に定める定理であって、m そのものの選び方までを一意化するものではない。B4 の正則性条件は可測性に緩めてもよく、病的な非可測解を避けるには連続性または可測性のいずれかがあれば足りる。q-加法性に基づく Tsallis 型の拡張もありうるが、本稿の立場は、厳密加法性こそが累積損失という概念を最も自然に支える要請である、というものである。前版で触れた q-対数の具体式やその比較は、最小形式としての見通しを優先して本稿からは意図的に省き、別途の補論的検討に委ねる。

4. 残存可能性の指数表現

命題 1.
  V^(0) ⊇ V^(1) ⊇ ... ⊇ V^(n)
を有限測度 m の下での可測集合列とし、
  l_i = -ln( m(V^(i)) / m(V^(i-1)) )
  L_n = Σ_{i=1}^n l_i
と定める。このとき
  m(V^(n)) = m(V^(0)) e^{-L_n}
が成り立つ。

証明.
定義より、
  e^{-l_i} = m(V^(i)) / m(V^(i-1))
であるから、
  e^{-L_n}
    = Π_{i=1}^n e^{-l_i}
    = Π_{i=1}^n m(V^(i)) / m(V^(i-1))
    = m(V^(n)) / m(V^(0)).
よって
  m(V^(n)) = m(V^(0)) e^{-L_n}.
証明終。

したがって指数型は追加仮定ではなく、§3 の公理系 B1–B4 から強制された対数比損失の望遠鏡積として恒等的に成り立つ。言い換えれば、指数型は損失測度の公理系と測度の乗法性から二段階で導かれる必然的な形式である。望遠鏡積による指数表現自体は既知の数学的構造に属するが、本稿の寄与は、その骨格を構造維持可能性の縮小という対象に与える点にある。経験的内容は、V^(i) と m の具体化、および各制約に対応する縮小率の差異に属し、検証は別稿に委ねる。


5. 構造持続ポテンシャル

命題1は、構造を維持できる状態集合の残存量がどのように表されるかを与える。しかし、構造の持続は、その残存量だけで決まるとは限らない。実際の系では、その構造を保つために利用できる資源もまた重要である。そこで、資源要因と構造維持可能性の縮小要因とを切り分けて表す最小の定式化として、命題1で与えた指数表現とは別に、有効維持資源を
  M > 0
として導入し、構造持続ポテンシャル S を
  S = Me^{-L_n}
と定める。これは命題1の直接の帰結ではなく、資源要因を組み込んだ拡張的定義である。

M は m(V^(0)) と同一視される必要はない。m(V^(0)) は構造維持可能状態の初期的広がりであるのに対し、M は系がその構造を維持するために利用できる有効資源の大きさを表す。個別の系において、それはエネルギー、時間、冗長性、回復可能性、計算余地、あるいは流れを維持する能力として具体化されうる。本稿では、その内訳には立ち入らない。

最も単純には、構造の持続はある閾値 S_c に対して
  S < S_c
となるとき失われる。このとき観測される現象は、崩壊、機能不全、相転移、自己同一性の喪失など、系に応じて異なる形をとる。

ここで失われるのは、基体の存在そのものではなく、その構造としての同一性や機能である。したがって、閾値を下回った系は、必ずしも消滅するとは限らず、別の構造として存続することがある。

6. 結論

構造の持続は、単なる資源残量の問題ではない。構造維持条件と両立する状態がなお存在するかどうかの問題でもある。本稿では、その事実を表す最小形式を与えた。構造を維持できる状態集合の逐次縮小に対して、損失測度に関する自然な公理系（比率依存性、正規化、加法性、連続性）が対数比の関数形を一意に強制し、その望遠鏡積として残存可能性の指数表現が恒等式として従う。指数型は仮定ではなく、公理系と測度の乗法性から二段階で導かれる帰結である。さらに、有効維持資源を導入することで、構造持続ポテンシャル
  S = Me^{-L}
を得た。この形式は、資源の不足と構造を維持できる状態の減少とを切り分けつつ、基体の存在そのものではなく、その構造としての持続の喪失を一つの形式で記述する。

本稿が与えるのは、個別理論に先立つ条件つきの最小骨格である。測度の選択、資源量の定義、閾値の同定、制約生成過程の安定性、実証、および応用は別稿に譲る。

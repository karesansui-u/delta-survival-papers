2_構造持続の条件つき導出
構造持続の条件つき導出
— 三つの条件、弱依存下の指数境界、形式検証 —

要旨

本稿の目的は、構造持続の指数表現が、どの条件のもとで導かれるかを明示することである。最小形式の中心式は薄いが、その背後には条件つきの導出がある。本稿では、A1–A3 を明示し、そのもとで指数型が導かれることを述べる。さらに、第三条件 A3 が厳密にではなく近似的にのみ成立する場合にも、結論が対応する誤差のもとで保たれることを示す。これらの導出は Lean により形式検証されている。本稿の役割は、最小形式の背後にある数学的骨格を、必要最小限の形で与えることにある。


1. はじめに

最小形式の中心式
  S = Me^{-L}
は、構造持続を記述するうえで十分に薄く、また広い応用可能性を持つ。しかし、その薄さのために、読者には次の点が見えにくいことがある。すなわち、この指数型はどこまでが定義の帰結であり、どこからが追加の条件に依存するのか、という点である。

本稿の目的は、この点を明示することである。ここで行うのは、最小形式そのものを拡張することではない。そうではなく、その背後にある条件つきの導出を、必要最小限のかたちで切り出すことである。

以下では、まず A1–A3 という三つの条件を置く。ついで、それらのもとで指数型が導かれることを述べる。さらに、A3 が厳密に成立しない場合にも、近似的に成立する範囲では結論が安定に保たれることを示す。最後に、これらの導出が Lean により形式検証されていることを記す。


2. 三つの条件

本稿で用いる条件は、次の三つである。

A1. 各段階で加わる制約は、構造を維持できる状態の集合を広げず、縮めるか、そのままに保つ。

A2. 各段階の損失は、その段階で生じた縮小率の対数比として測られる。

A3. 各段階の損失は、先行する段階の損失から独立である。

A1 は、制約が縮小方向に働くという条件である。A2 は、損失の測り方を定める条件である。A3 は、各段階の寄与が互いに独立に足し合わされるための構造的条件である。

A2 と A3 は密接に関連している。乗法的な縮小を対数空間で記述すること（A2）と、その対数的寄与が加法的に累積すること（A3）は、同一の表現選択の二つの側面である。しかし、A3 を独立性・弱依存の条件として切り出すことには、近似的成立を論じる上で意味がある（4節参照）。

これらは強い一般理論を与えるための条件ではない。とくに A3 は、実系一般でそのまま成立する経験法則としてではなく、条件つき導出を切り出すための明示的仮定として導入されている。厳密独立が失われた場合に理論が無意味になるのではなく、主張が等式から境界へ、あるいは実証的傾向へと弱まることを示すのが、本稿の目的である（4節参照）。


3. 条件つきの指数表現

A1–A3 のもとでは、構造を維持できる状態の残存量は、累積損失に対して指数型で表される。すなわち、初期残存量を m(V^(0))、累積損失を L とすると、
  m(V^(n)) = m(V^(0)) e^{-L}
という形が得られる。

重要なのは、この指数型が別個の仮定として導入されるのではないことである。それは、縮小率を対数比で測り、その寄与が加法的に累積するという条件から直接に現れる。したがって、指数型は経験的に当てはめた関数形ではなく、この条件つきの枠組みの帰結である。

最小形式における S = M exp(-L) は、この残存量の指数表現に、有効維持資源 M を別に導入した拡張的定義である。

この意味で、本稿が示すのは、最小形式の指数型が「なぜその形になるのか」を支える骨格である。


4. 近似条件のもとでの安定性

現実の系では、A3 の意味での厳密な独立が常に成立するとは限らない。以下では、A3 を弱依存の条件へ緩和した場合を考える。段階ごとの損失がわずかな相互作用やずれを含むとき、累積損失は厳密な意味での単純な和から少しずれる。

A1–A3 が厳密に成立する場合には、3節の通り、S = S_0 exp(-L) が等号として成り立つ。

さらに、A3 が厳密な独立としては成立せず、弱い依存のみがある場合にも、残存可能性は指数型の範囲に収まる。具体的には、ある段階の損失が先行する段階の損失にどれだけ引きずられるかを ρ（|ρ| < 1）で表す。ρ = 0 は完全な独立、ρ > 0 は先行段階の損失が次の段階の損失を増幅する方向の依存、ρ < 0 は減衰する方向の依存を意味する。このとき、残存可能性 P は
  e^{-L(1+ρ)} ≤ P ≤ e^{-L(1-ρ)}
に挟まれる。ρ → 0 のとき等号に戻り、ρ が有限でも、指数型の形は保たれる。

したがって、A3 の近似版のもとでも、指数型は崩れない。完全な独立が成立しなくても、弱い依存の範囲では結論は安定に保たれる。

この点は重要である。なぜなら、最小形式を実際の系に適用する際には、厳密な独立性そのものよりも、その近似的成立が意味を持つ場合が多いからである。


5. 形式検証

本稿で述べた A1–A3 のもとでの指数表現の導出、および A3 の近似版に対する対応する結果は、Lean により形式検証されている。

ここで形式検証の対象となっているのは、条件つきの数学的導出そのものである。したがって、最小形式の中心的な導出は、少なくともこの条件の範囲内では、機械的に検証可能な形に置かれている。

検証コードは https://github.com/karesansui-u/delta-survival-papers/tree/main/lean において公開されている。検証対象は以下の通りである。
- AxiomsToExp.lean: A1–A3 のもとでの指数表現の導出
- WeakDependence.lean: A3 を弱依存に緩和した場合の ρ-付き指数境界
- RobustSurvival.lean: 弱依存条件のもとでの対応する拡張結果

他方で、形式検証は、個別領域における測度の選択、資源量の具体化、経験的妥当性、あるいは応用の成功そのものまでを保証するものではない。本稿で確認されるのは、条件を明示したうえでの導出の論理的一貫性である。


6. 限界

本稿は、構造持続のすべてを一般的に証明するものではない。また、あらゆる系において A1–A3 がそのまま成立すると主張するものでもない。

本稿が与えるのは、より限定された主張である。すなわち、A1–A3 が成立するか、あるいは少なくとも A3 が近似的に成立するならば、最小形式の指数表現はその範囲で正当化される、ということである。

したがって、本稿の役割は、応用の全体を与えることではなく、最小形式の背後にある条件つきの数学的骨格を、できる限り薄く与えることにある。


7. 結論

本稿では、構造持続の指数表現が、A1–A3 という三つの条件のもとで導かれることを述べた。ここで重要なのは、指数型が追加仮定ではなく、縮小率の対数的記述と損失の加法的累積から直接に現れることである。さらに、A3 を弱依存へ緩和した場合にも、残存可能性は
  e^{-L(1+ρ)} ≤ P ≤ e^{-L(1-ρ)}
の形の指数境界に挟まれる。したがって、完全な独立が成り立たない場合にも、指数型は安定に保たれる。

このことにより、最小形式は単なる見通しのよい表現にとどまらず、条件を明示すれば、形式的に支えられた骨格を持つことがわかる。本稿の役割は、その骨格を必要最小限のかたちで示すことにある。
